About This Project

This project was created as a portfolio example to demonstrate API integration, JSON processing, dynamic DOM updates, responsive layout design, and practical JavaScript development.

Technologies Used

The dashboard is built with HTML for structure, CSS for styling, and JavaScript for fetching and displaying live cryptocurrency data from the CoinGecko public API.

Project Goal

The goal is to build a simple but professional-looking dashboard that shows how front-end code can connect to public APIs and present useful market information.
