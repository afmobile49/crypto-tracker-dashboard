Crypto FAQ

Some basic questions about cryptocurrency and this small project.

What is cryptocurrency?

Cryptocurrency is a digital form of money that uses cryptography and blockchain technology.

What is blockchain?

Blockchain is a distributed database that stores transaction records across many computers.

Where does this project get the data?

Market data is retrieved from the public CoinGecko API.
