Frequently Asked Questions

This page explains basic cryptocurrency concepts and how this dashboard project works.

What is cryptocurrency?

Cryptocurrency is a digital form of money that uses cryptography for security. It is usually not controlled by a single central authority like a bank or government.

What is blockchain?

Blockchain is a distributed digital ledger that records transactions across many computers. It helps provide transparency, security, and resistance to unauthorized changes.

Why are cryptocurrencies important?

Cryptocurrencies can support global payments, fast transfers, and new financial systems such as decentralized finance. They also make it possible to move digital value without traditional intermediaries.

Which coins are shown in this dashboard?

  • Bitcoin (BTC)
  • Ethereum (ETH)
  • Solana (SOL)
  • Tether Gold (XAUT)

How does this project work?

The dashboard sends a request to a public cryptocurrency API, receives JSON data, and then uses JavaScript to display the live price, daily change, and market cap on the page.

Which API is used?

This project uses the CoinGecko public API to retrieve market data for selected cryptocurrencies.
