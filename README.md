

\## How It Works



1\. The page loads and JavaScript runs automatically.

2\. The script sends a request to the CoinGecko API.

3\. The API returns cryptocurrency data in JSON format.

4\. JavaScript processes the data.

5\. The dashboard updates the cards and table with live prices.



\## How to Run the Project



1\. Download or clone the repository.

2\. Open `index.html` in a web browser.

3\. The dashboard will automatically load live market data.



You can also click the \*\*Refresh Data\*\* button to reload the information.



\## Example Data Displayed



\- Bitcoin (BTC)

\- Ethereum (ETH)

\- Solana (SOL)

\- Tether Gold (XAUT)



Each coin displays:



\- Name

\- Symbol

\- Current price in USD

\- 24-hour price change

\- Market capitalization
